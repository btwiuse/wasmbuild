# wasmbuild starter

![cover](https://deno.com/wasmbuild/cover.png)

## build

```
$ make
```

## run in deno

```
$ make deno
```

## run in browser

```
$ make browser
```

## run tests

```
$ make test
```

## view docs

```
$ make docs
```
